## Usage

Each vibe file contains:
- **Overview**: Keywords, best-for, inspiration sources
- **Design Philosophy**: The "why" behind the vibe
- **Color Palette**: Complete hex codes with usage notes
- **Typography**: Font families, sizes, weights
- **Spacing & Layout**: Container widths, padding, grid systems
- **Animation Philosophy**: Entrance and interaction animations
- **Component Styles**: CSS for buttons, cards, navigation
- **Section Layout Guidelines**: Hero, About, Projects, Skills, Contact
- **Do's and Don'ts**: Quick reference for maintaining the vibe
